6. Analytics (Umami)

This website uses Umami, a privacy-friendly, open-source analytics tool that is self-hosted on our own infrastructure. Umami does not use cookies, does not collect personal data, and does not track users across websites. It collects only anonymous aggregate data (e.g. page views, referrer, device type, country). No consent is required for this processing, as no personal data is collected.

To exercise any of these rights, contact: connect@lourensvanderzee.com
